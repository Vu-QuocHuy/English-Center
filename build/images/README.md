# Hướng dẫn thêm hình ảnh cho Landing Page

Để Landing Page hoạt động tốt, vui lòng thêm các hình ảnh sau vào thư mục này:

1. `ielts.jpg` - Hình ảnh cho khóa học IELTS

   - Kích thước đề xuất: 800x600px
   - Nội dung: Hình ảnh liên quan đến luyện thi IELTS

2. `toeic.jpg` - Hình ảnh cho khóa học TOEIC

   - Kích thước đề xuất: 800x600px
   - Nội dung: Hình ảnh liên quan đến luyện thi TOEIC

3. `kids.jpg` - Hình ảnh cho khóa English for Kids
   - Kích thước đề xuất: 800x600px
   - Nội dung: Hình ảnh trẻ em học tiếng Anh

## Lưu ý:

- Sử dụng hình ảnh chất lượng cao
- Đảm bảo bạn có quyền sử dụng hình ảnh
- Tối ưu kích thước file để trang web tải nhanh
- Đặt tên file chính xác như đã nêu ở trên
